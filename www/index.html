Documentation

1\ Upload a track into Gene Browse

The required format for anotation tracks is GFF3.

To upload a GFF into GBrowse, go to page fi.muni.cz/ , click …

NOTICE: GBrowse will accept any file without complaints, although if the file is malformed, some annotations will be displayed wrong or will be missing completely. Therefore it is advised to validate the file first.

2\ Validate a GFF3 file

One can validate a GFF3 file by uploading it on an online validating webpage or by downloading and running a validating program on their computer

Off-line tools

This section describes how to install GenomeTools gff3validator

You can get GenomeTools at this linked website. Be sure to download a binary distribution as the process of building the software from source is unnesessary and tedious. If there is several download possibilities for your operating system, choose the complete version, but the barebone one should work also?

extract the archive

In Linux, right click the file in nautilus, dolphin, krusaider, mc or any other file manager and select Extract. In Bash, enter command like (you may need to customize it) 
tar -C /path/to/install -xzf downloadedfile.tar.gz

explanation of the switches

 -C, --directory=DIR
              change to directory DIR
      -x, --extract, --get
              extract files from an archive
              
       -z, --gzip
              filter the archive through gzip

-f, --file=ARCHIVE
              use archive file or device ARCHIVE

validating

run /path/to/install/gt gff3validator jmenosouboru.gff to perform basic validation.
